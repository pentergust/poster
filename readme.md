# Poster

> Напишите одно сообщение, отправьте его везде.

Суть проекта очень простая:

1. Вы пишете сообщение в формате TOML + MarkDown.
2. Настраиваете источники отправки.
3. Отправляете сообщение в выбранные источники.

> [!WARNING]
> Проект находится в ранней стадии разработки и не готов к использованию.
> Пожалуйста ознакомьтесь с [TODO](./todo.md) для оценки функционала.

## Сборка

Всё довольно просто:

```sh
git clone https://github.com/pentergust/poster
```

Сборка проекта из исходного кода:

```sh
go build -ldflags="-s -w" -o poster cmd/main.go
```
