# Poster to-do

Как обстоят дела с функционал.

## Функционал

Что должна уметь программа в общем.

- [X] Отправка сообщения.
- [ ] Валидация сообщения.
- [ ] Web сервер для приёма сообщений.

## Message

Полнота отправляемого сообщения.

- [X] `Content`: Он же текст сообщения в формате MarkDown.
- [ ] `Embed`: Более полное сообщение.

## Telegram

Отправка сообщения через Telegram бота пользователям, чатам, каналам.

- [X] Отправка сообщения.
- [X] Markdown форматирование.
    Стоит заметить что MarkDown **не полный**. Некоторые элементы форматирования
    не поддерживаются.
    Как например заголовки.
- [ ] Embed сообщения

## Discord

Отправка сообщение через Discord webhook.

- [X] Отправка сообщения.
- [X] Markdown.
- [ ] Создание/отправка сообщения в форум.
- [ ] Embed.
- [ ] Кнопки.


## Web

Генерация страниц сайта.
К примеру для блогов.

> Пока в разработке, планируется в будущем.
